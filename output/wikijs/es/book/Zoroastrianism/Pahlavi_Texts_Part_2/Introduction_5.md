## ABREVIATURAS UTILIZADAS EN ESTE VOLUMEN.

Acc. para caso acusativo; Af. Zarat. para Âfringân-i Zaratû<i>s</i>t; anc. Pers. para persa antiguo; App. para apéndice a este volumen; Ar. para árabe; AV. para el Libro de Ar<i>d</i>â-Vîrâf, ed. Hoshangji y Haug; Av. para Avesta; B29 para persa Rivâyat MS. No. 29 de la Biblioteca Universitaria de Bombay; Bd. y Byt. para Bundahi<i>s</i> y Bahman Vast, como se traduce en el vol. v de esta serie; BK para una antigua copia imperfecta de K35 escrita en Kirmân, pero ahora en Bombay; Chald. para caldeo; comp. para comparar; Dd. para Dâ<i>d</i>istân-î Dînîk, como se traduce en este volumen; Dk. para Dînka<i>r</i><i>d</i>; Ep. para Epístolas de Mânû<i>s</i><i>k</i>îhar, como se traduce en este volumen; Farh. Okh. para Farhang-i Oîm-khadûk, ed. Hoshangji y Haug; Gen. para Génesis; Ensayos de Haug para Ensayos sobre la lengua sagrada, escritos y religión de los parsis, por M. Haug, 2.ª edición; Hn. para Hâ<i>d</i>ôkht Nask, como se publicó con AV.; Huz. para Huzvâri<i>s</i>; Introd. para Introducción; J. para Dd. MS. perteneciente a Dastûr Jâmâspji Minochiharji en Bombay; K35 para Dd. MS. N.º 35 de la Biblioteca Universitaria de Copenhague; L4 para Vend. MS. N.º 4 de la India Office Library en Londres; Lev. para Levítico; M5, M7, M10, M14 para MSS. Núms. 5, 7, 10, 14 de la Colección Haug en la Biblioteca Estatal de Múnich; Mkh. para Mainyô-i-khard, ed. West; n. para nota a pie de página; nom. para caso nominativo; p. para página; Pahl. para Pahlavi; Pâz. para Pâzand; Pers. para Persa; pp para participio pasado; Sîr. para Sîrozah, ed. Westergaard; Sls. para Shâyast-lâ-shâyast, como se traduce en el vol. v de esta serie; TD para Bd. MS. perteneciente al Sr. Tehmuras Dinshawji en Bombay; TK para Dd. MS. perteneciente al mismo; trans. D. para traducido por Darmesteter, en el vol. iv de esta serie; Vend., Visp. y Yas. por Vendidâ<i>d</i>, Vispara<i>d</i> y Yasna, ed. Spiegel; Yt. por Ya<i>s</i>t, ed. Westergaard; Z<i>D</i><i>M</i><i>G</i>. por Zeitschrift der deutschen morgenländischen Gesellschaft; Zs. por Selections of Zâ<i>d</i>\-sparam, tal como se traduce en el vol. v de esta serie.
