### CAPÍTULO III.

§ 1. En un hogar doble, se pueden colocar [^86] alimentos cocinados poco antes del Sabbath si se calienta con rastrojo o leña; pero si se calienta con pepitas de olivo o leña, no se deben colocar alimentos hasta que el compartimento inferior esté limpio o cubierto de ceniza. Beth Shammai sostiene que solo se puede colocar agua caliente, pero no alimentos cocinados, en un hogar o hogar calentado de esta manera; pero Beth Hillel sostiene que se pueden colocar tanto agua caliente como alimentos cocinados. Bet Shamai sostiene que pueden sacar cualquier cosa del hogar, pero no deben volver a colocarla allí; pero Bet Hillel sostiene que también pueden volver a colocarla.

§ 2. Si un horno se calienta con rastrojo o maleza, no se debe poner nada ni dentro ni encima. Un hogar sencillo calentado con rastrojo o maleza se considera como uno doble; pero si se calienta con aceitunas o leña, se considera como un horno.

§ 3. No se debe poner un huevo al lado de una olla caliente \[en sábado\], para que se asiente \[cocido\], ni se debe envolver en <span id="p42">[<sup><small>p. 42</small></sup>]</span> paños calientes \[con esa intención\], ni se debe poner en arena caliente o polvo del camino\[para que el huevo\] se tueste \[por el calor del sol\].

§ 4. Sucedió una vez que los habitantes de Tiberíades llevaban una tubería de agua fría a través de un arroyuelo de su pozo caliente; pero los sabios les explicaron que en sábado, esta agua, como cualquier otra que se haya calentado en sábado, está prohibida tanto para lavar como para beber; y que en las festividades, esta agua, como cualquier otra que se haya calentado en la festividad, está prohibida para lavar, pero se permite beber. Un מוליאר [^87] que ha sido purificado de carbones, es lícito beber de él en sábado; un אנטיכי [^88], aunque haya sido purificado de carbones, no es lícito beber de él.

§ 5. En la olla con agua caliente, retirada del fuego, no se debe verter una pequeña cantidad de agua fría para calentarla, pero sí se puede verter agua fría en una olla o vaso con agua tibia para entibiarla. 89. En una olla o tetera retirada del fuego, no se deben añadir especias; pero sí se pueden añadir especias a los alimentos calientes en un plato o fuente. R. Jehudah dice: «En todo tipo de recipientes se pueden añadir especias, excepto en los alimentos que contengan vinagre o salvado».

§ 6. No es lícito colocar un recipiente debajo de una lámpara para recoger el aceite que gotea, pero si se ha colocado así durante el día anterior al inicio del Sabbath, se permite dejarlo; pero el aceite que se haya recogido no debe usarse, ya que no forma parte de lo preparado para el consumo del Sabbath. Se puede mover una lámpara nueva, pero no una vieja que ya se haya usado. R. José dice: «Se pueden mover todas las lámparas, excepto las que están encendidas para el Sabbath». Se puede colocar un recipiente debajo de una lámpara para recoger las chispas, pero no se debe echar agua, ya que se apaga.

## Notas al pie

[^86]: 41:1 Un hogar \[horno o estufa\] dividido en dos compartimentos: en el inferior se pone el combustible, y en el superior hay espacio para dos platos.

[^87]: 42:2 Una cacerola para calentar, con el receptáculo para las brasas a un lado; el calor de una cacerola así es mucho menor que el de un ‏אנטיכי‎.

[^88]: 42:3 Una cacerola doble para calentar, con el receptáculo para las brasas entre las dos cacerolas.

[^89]: 42:4 Todas las regulaciones a lo largo de este Tratado se refieren únicamente al sábado, a menos que se indique expresamente lo contrario.
