### CAPÍTULO VIII.

§ 1. Quien lleve suficiente vino para llenar una copa, [^114] [como el que se usa para bendecir después de la comida], leche equivalente a un trago, miel suficiente para untar una llaga, aceite suficiente para ungir una pequeña articulación, agua suficiente para hacer un colirio, o cualquier otra bebida, un cuarto de litro, y también de cualquier líquido que pueda verterse, un cuarto de litro. R. Simeón dice: «Para todos los artículos aquí enumerados, la cantidad legal es un cuarto de litro; y las [diversas] cantidades aquí especificadas son solo para quienes conservan los artículos».

§ 2. Quien lleve cuerda suficiente para hacer un asa para una caja o una cesta, o caña suficiente para hacer un gancho para colgar un tamiz fino o grueso, R. Jehudah dice: «Lo suficiente para medir el zapato de un niño, papel suficiente para escribir en él una factura de peaje [^115], y, por último, quien lleve una factura de peaje de papel seco suficiente para envolver en ella un pequeño frasco de bálsamo, es culpable».

§ 3. Pergamino suficiente para un amuleto, pergamino de doble trama suficiente para escribir una mezoozah sobre él, pergamino de trama simple suficiente para escribir sobre él la sección más pequeña de la parashá del Tefilín, que es ‏שמע ישראל‎, tinta suficiente para escribir dos letras, pintura suficiente para teñir un ojo. [^116]

§ 4. Cal de ave, suficiente para cubrir la parte superior de la rama de tilo; brea o azufre, suficiente para llenar un tubo de azogue; cera suficiente para tapar una pequeña fuga en un barril de vino; arcilla suficiente para hacer un orificio para un fuelle, o el crisol de un refinador de oro. R. Jehudah dice: «Suficiente para hacer un soporte para tal crisol; cal suficiente para cubrir el dedo meñique de una doncella». R. Jehudah dice: «Suficiente para cubrir una de sus sienes». R. Nehemías dice: «Suficiente para ungir una de sus sienes». [^117]

§ 5. Tierra roja suficiente para sellar un fardo de mercancías. Tal es el dictamen de Rabí Akivah; pero los sabios sostienen: «suficiente para sellar una carta con ella». Estiércol o arena fina suficiente para abonar un tallo de col. Tal es el dictamen de Rabí Akivah; pero los sabios sostienen: «suficiente para abonar un puerro con ella». Arena gruesa suficiente para llenar una paleta, caña suficiente para hacer una pluma, y ​​si es gruesa o se parte, suficiente para hervir el huevo más ligero [^118], que, mezclado con aceite, se encuentra en una cáscara caliente.

§ 6. Hueso suficiente para hacer una cuchara. R. Jehudah dice: «Suficiente para hacer un encaje para una llave». Vidrio suficiente para raspar la punta de una lanzadera de tejedor, un trozo de moho o una piedra lo suficientemente grande como para lanzarla a un pájaro. R. Eleazar ben Jacob dice: «Suficientemente grande como para lanzarla a una bestia».

§ 7. Tiestos lo suficientemente grandes como para colocarlos entre los manojos y equilibrarlos. Tal es el dictamen de R. Jehudah; pero R. Meir dice: «Lo suficientemente grandes como para recoger con ellos el combustible encendido». R. José dice: «Lo suficientemente grandes como para contener un cuarto de un lingote». R. Meir dice: «Aunque no hay ninguna prueba absoluta que respalde mi afirmación, sí hay indicios de mi afirmación; pues se dice: «No se encontrará al romperlo un tiesto que pueda encender el hogar». [^119] Pero R. José respondió: «¿Puedes aportar una prueba? ¿No concluye la frase con las palabras «o para sacar agua del pozo»?» [^120]

## Notas al pie

[^114]: 48:1 La copa contiene un cuarto de litro. La proporción al mezclar es tres partes de agua por una de vino. Por lo tanto, la cantidad a la que se refiere aquí es un dieciseisavo de litro.

[^115]: 48:2 La factura del peaje contenía dos letras de gran tamaño.

[^116]: 48:3 En Oriente es costumbre que las mujeres se tiñan las pestañas y se pinten los párpados con pintura preparada a partir de una raíz llamada henna.

[^117]: 49:4 Es una costumbre oriental aplicar un ungüento de cal en la frente, como cosmético para suavizar la piel.

[^118]: 49:5 El huevo de gallina doméstica se considera el más ligero y el que se cuece más rápido.

[^119]: 49:6 Isaías xxx. 14.

[^120]: 49:7 Isaías xxx. 14.
