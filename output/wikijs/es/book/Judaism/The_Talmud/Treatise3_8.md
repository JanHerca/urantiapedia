<span id="p93"><sup><small>[ p. 93 ]</small></sup></span>

1\. Se puede servir vino suficiente para la copa, [^117] leche suficiente para un trago, miel suficiente para una contusión, aceite suficiente para ungir un miembro pequeño, agua suficiente para humedecer el colirio, y el resto de las bebidas un cuarto de log, y lo que se pueda verter [^118] un cuarto de log. Rabí Simeón dice: «Todos por un cuarto de log». Y no mencionaron estas medidas, salvo para quienes las reservan.

2\. «¿Quien saque suficiente cuerda para hacer una oreja para una tina, suficiente junco para colgar el cedazo y el acertijo?», dijo el rabino Judah, «suficiente para tomar de ella la medida del zapato de un niño; suficiente papel para escribir en él la firma de los recaudadores de impuestos; suficiente papel borrado para envolver una pequeña botella de bálsamo, es culpable» (de muerte).

3\. Cuero suficiente para un amuleto; pergamino pulido por ambos lados, suficiente para escribir un letrero para un poste de puerta; vitela suficiente para escribir en él una pequeña porción, que está en filacterias, es decir, «Escucha, Israel»; tinta suficiente para escribir dos letras; kohl [^119] suficiente para pintar un ojo.

4. Cal de ave, suficiente para colocar en la parte superior de una percha; brea o azufre para rellenar un agujero; cera, suficiente para rellenar la boca de un agujero pequeño; arcilla de ladrillo, suficiente para hacer la boca de un fuelle de crisol para orfebres (el rabino Judah dice: «suficiente para hacer un soporte de crisol»); salvado, suficiente para colocar en la boca de un soplete de crisol para orfebres; ungüento, suficiente para ungir el dedo meñique de las niñas (el rabino Judah dice: «suficiente para hacer crecer el cabello»); el rabino Nehemías dice: «para refrescar el templo».

5\. Tierra roja, «como sello de mercaderes», según Rabí Akiba; pero los Sabios dicen: «como sello de cartas»; estiércol y arena fina, «suficientes para abonar un tallo de col», según Rabí Akiba; pero los Sabios dicen: «suficientes para abonar un puerro»; arena gruesa, suficiente para construir una calera; una caña, suficiente para hacer una pluma. «¿Pero si es <span id="p94"><sup><small>[ p. 94 ]</small></sup></span> gruesa o partida?», «suficiente para cocer con ella un huevo de gallina fácil de cocer entre huevos, mezclado con aceite y puesto en una sartén».

6\. Un hueso suficiente para hacer una cuchara —dijo el rabino Judah—, suficiente para hacer la guarda de una llave; un cristal suficiente para raspar la parte superior de una lanzadera; un trozo de tierra o una piedra suficiente para lanzarla a un pájaro; dijo el rabino Eliezer, «suficiente para lanzarla a una bestia».

7\. «¿Un tiesto?» «Suficiente para poner entre dos vigas», palabras del rabino Judah; el rabino Meier dice: «Suficiente para apagar el fuego con él»; el rabino José dice: «Suficiente para contener la cuarta parte de un leño». El rabino Meier dijo: «Aunque no hay prueba visible del asunto, sí hay indicios, como se dice: “No se encontrará al romperlo un tiesto que apague el fuego del hogar». [^120] El rabino José le respondió: «Ahí está la prueba visible: 'o para sacar agua del pozo'». [^120]

_Los dos tercios restantes de este tratado no se traducen, por carecer de interés y contener partes no aptas para su publicación_.
