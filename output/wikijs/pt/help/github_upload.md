Para enviar conteúdo para Urantiapedia, é usado um projeto GitHub com um backup completo do site [Urantiapedia](https://urantiapedia.org). O projeto está aqui:

[Urantiapedia-backup](https://github.com/JanHerca/urantiapedia-backup)

Este projeto está sincronizado com o site para que uma cópia completa de todo o conteúdo de *Urantiapedia* possa ser recuperada.

Para enviar conteúdo, siga estas etapas:
- Clone ou fork o projeto *Urantiapedia-backup* no GitHub.
- Copie o novo conteúdo na pasta `output/wikijs` do projeto *Urantiapedia* para a pasta raiz no projeto *Urantiapedia-backup*.
- Faça um pull request.
- Os usuários *Admin* do projeto GitHub mesclarão as alterações. Assim que as alterações forem mescladas, elas estarão disponíveis nas próximas horas na página da web *Urantiapedia* dependendo do desempenho atual nos servidores para executar trabalhos de upload.
