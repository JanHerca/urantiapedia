Aqui está como criar um catálogo de diagramas do _O Livro de Urântia_.

## Processo

O processo é idêntico ao explicado para os artigos. Consulte [Criando um catálogo de artigos](/pt/help/github_articles_catalog). A lista provisória de diagramas pode ser encontrada aqui: [Diagramas planejados](/en/index/diagrams).

Para editar essa lista, você pode fazer isso de duas maneiras:

- Faça login aqui em _Urantiapedia_. Vá para a lista de diagramas: [Diagramas planejados](/en/index/diagrams). Coloque a página em edição no menu _Page actions > Edit_ no painel de botões superior direito. Um editor Markdown simples será aberto, onde você poderá editar a listagem. (IMPORTANTE: para poder fazer isso, você precisa de um Administrador para habilitar as permissões de edição nessa página.)
- No projeto GitHub modifique este arquivo: `/output/wikijs/en/index/diagrams.md`. Para modificá-lo, você pode simplesmente baixar esse arquivo ou todo o projeto do GitHub. Veja a [Ajuda dos Editores Assistentes](/pt/help/github_assistant) para saber como baixar coisas do GitHub.

Em qualquer um dos dois casos anteriores, o arquivo a ser editado é um arquivo Markdown muito simples onde para cada diagrama deve ser incluído o seguinte, na forma de uma tabela:

- Um sinalizador `:white_square_button:` indicando que o diagrama ainda não foi criado.
- Título.
- O autor.
- Alguns rótulos opcionais que categorizam o diagrama.
- O URL de onde esse diagrama pode ser baixado.

```
| Status                | Title                              | Author  | Tags  | URL                               |
| ----------------------| -----------------------------------| --------| ----- | --------------------------------- |
| :white_square_button: | "The Kingdom of Heaven Revelation" |         |       | http://www.universalfather.info/  |
```

## Enviando arquivos

Se você modificou a listagem do arquivo no GitHub, você tem duas opções para confirmar suas alterações: 1) se você for um “Editor Chefe” use o GitHub conforme detalhado na [Ajuda do Editor Chefe](/pt/help/github) ; se você for um “Editor Assistente”, você pode simplesmente enviar esse arquivo Markdown modificado para o e-mail de qualquer Administrador ou para urantiapedia@gmail.com.
