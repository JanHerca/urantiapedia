Pour valider le contenu sur Urantiapedia, un projet GitHub est utilisé avec une sauvegarde complète du site Web [Urantiapedia](https://urantiapedia.org). Le projet est ici :

[Urantiapedia-backup](https://github.com/JanHerca/urantiapedia-backup)

Ce projet est synchronisé avec le site Web afin qu'une copie complète de tout le contenu d'*Urantiapedia* puisse être récupérée.

Pour valider le contenu, procédez comme suit :
- Clonez ou bifurquez le projet *Urantiapedia-backup* dans GitHub.
- Copiez le nouveau contenu dans le dossier `output/wikijs` du projet *Urantiapedia* dans le dossier racine du projet *Urantiapedia-backup*.
- Faites une pull request.
- Les utilisateurs *admin* du projet GitHub fusionneront les modifications. Une fois les modifications fusionnées, elles seront disponibles dans les prochaines heures sur la page Web *Urantiapedia* en fonction des performances actuelles des serveurs pour exécuter les tâches de téléchargement.
