<small>Une note écrite par une main ultérieure, copiée d'une autre écriture.</small>

**(125)**(Voici les noms que je donnerai à partir de l'Éternel. Écris-les avec un signe, afin que les Fils de Dieu soient révélés dès maintenant.

Voici le nom de l'Immortel : _aaa_, _ōōō_ ; et voici le nom de la Voix pour laquelle l'Homme Parfait s'est mis en mouvement : _iii_. Voici l'interprétation des noms de ces mystères : le premier nom, qui est _aaa_, s'interprète comme _fff_ ; le second, qui est _mmm_ ou ōōō, s'interprète comme _aaa_ ; le troisième, qui est _ps ps ps_, s'interprète comme _ooo_ ; le quatrième, qui est _fff_, s'interprète comme _nnn_ ; le cinquième, qui est _ddd_, s'interprète comme _aaa_. Celui qui est sur le trône est _aaa_. Voici l'interprétation du second : _aaaa_, _aaaa_, _aaaa_ ; voici l'interprétation du nom entier.)
