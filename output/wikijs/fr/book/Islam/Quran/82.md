<span id="v1"><sup><small>1</small></sup></span>  Quand le ciel est fendu,
<span id="v2"><sup><small>2</small></sup></span>  Lorsque les planètes sont dispersées,
<span id="v3"><sup><small>3</small></sup></span>  Quand les mers se déversent,
<span id="v4"><sup><small>4</small></sup></span>  Et les sépulcres sont renversés,
<span id="v5"><sup><small>5</small></sup></span>  Une âme saura ce qu'elle a envoyé avant elle et ce qu'elle a laissé derrière elle.
<span id="v6"><sup><small>6</small></sup></span>  Ô homme, qu'est-ce qui t'a rendu insouciant envers ton Seigneur, le Généreux,
<span id="v7"><sup><small>7</small></sup></span>  Qui t'a créé, puis façonné, puis proportionné ?
<span id="v8"><sup><small>8</small></sup></span>  Dans la forme qu'Il veut, Il te jette.
<span id="v9"><sup><small>9</small></sup></span>  Non, mais vous niez le Jugement.
<span id="v10"><sup><small>10</small></sup></span>  Voici au-dessus de vous des gardiens,
<span id="v11"><sup><small>11</small></sup></span>  Généreux et enregistrant,
<span id="v12"><sup><small>12</small></sup></span>  Qui sait (tout) ce que vous faites.
<span id="v13"><sup><small>13</small></sup></span>  En vérité, les justes seront certainement dans le bonheur.
<span id="v14"><sup><small>14</small></sup></span>  Et voici, les méchants seront certainement dans l'enfer ;
<span id="v15"><sup><small>15</small></sup></span>  Ils y brûleront au Jour du Jugement,
<span id="v16"><sup><small>16</small></sup></span>  Et ne sera pas absent de là.
<span id="v17"><sup><small>17</small></sup></span>  Ah, qu'est-ce qui te dira ce qu'est le Jour du Jugement !
<span id="v18"><sup><small>18</small></sup></span>  Encore une fois, qu'est-ce qui te dira ce qu'est le Jour du Jugement !
<span id="v19"><sup><small>19</small></sup></span>  Un jour où aucune âme n'a de pouvoir sur aucune (autre) âme. Le commandement (absolu) de ce jour est celui d'Allah.
