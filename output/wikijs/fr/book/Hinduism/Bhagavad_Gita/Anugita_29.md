Brahman a dit :

Je dirai en vérité tout ce qui a un commencement, un milieu et une fin [^1751], ainsi que les moyens de le comprendre, et les noms et les caractéristiques [^1752]. Il est dit que le jour fut le premier, puis la nuit ; que les mois ont le brillant [^1753] en premier, les Nakshatras <i>S</i>ravana [^1754] comme le premier (parmi eux), et les saisons l'hiver comme le premier (parmi eux). La terre est la source [^1755] des odeurs, l'eau des saveurs, la lumière du soleil est la source des couleurs, le vent est dit être la source (des sensations du) toucher ; de même, l'espace est la source du son. Telles sont les qualités des éléments. Je vais maintenant énoncer les entités les plus élevées et les premières de toutes. Le soleil est <span id="p353"><sup><small>[ p. 353 ]</small></sup></span> le premier parmi les corps brillants [^1756] ; le feu est considéré comme le premier des éléments [^1757] ; Sâvitrî [^1758] de toutes les branches du savoir ; Pra<i>g</i>âpati des divinités ; la syllabe Om de tous les Védas ; et le Prâ<i>n</i>a, vent de vie, de tous les mots [^1759] ; tout ce qui est prescrit dans ce monde, tout cela est appelé Sâvitrî [^1760]. Le Gâyatrî est le premier parmi les mètres ; parmi les animaux (sacrificiels), la chèvre [^1761] est mentionnée (comme la première). Les vaches sont les premières parmi les quadrupèdes et les deux fois nées parmi les hommes [^1762]. Le <i>S</i>yena est le premier parmi les oiseaux ; parmi les sacrifices, l'offrande (au feu) est la meilleure ; et parmi tous les reptiles, ô le meilleur des deux fois nés ! le serpent [^1763] est le plus élevé. De tous les âges, le K<i>ri</i>ta est le premier, il n'y a aucun doute là-dessus. Parmi toutes les choses précieuses, l'or (est le premier), et parmi les végétaux (produits) également la graine d'orge [^1764]. Parmi toutes les choses à manger ou à avaler, la nourriture est la plus élevée ; et de <span id="p354"><sup><small>[ p. 354 ]</small></sup></span> toutes les substances liquides qui doivent être bues, l'eau est la meilleure. Et parmi toutes les entités immobiles, sans distinction, le Plaksha, le champ toujours saint de Brahman [^1765], est déclaré être le premier. Moi aussi, (suis le premier) parmi tous les patriarches [^1766], il n'y a aucun doute là-dessus. Et l'impensable, l'auto-existant Vish<i>n</i>u est déclaré être mon propre soi. De toutes les montagnes, le grand Meru est considéré comme le premier-né. Et parmi tous les quartiers et sous-quartiers, de même, le quartier oriental [^1767] est le premier. De même, le Gangâ, qui s'étend sur trois chemins, est considéré comme le premier-né parmi les rivières. Et de même, de tous les puits et réservoirs d'eau, l'océan est le premier-né. Et de tous les dieux, Dânavas, Bhûtas, Pi<i>s</i>â<i>k</i>as, serpents et Rakshases, et des hommes, Kinnaras et Yakshas, ​​Î<i>s</i>vara [^1768] est le seigneur. Le grand Vish<i>n</i>u, qui est empli du Brahman, et que nul être supérieur n'est dans ces trois mondes, est la source de tout l'univers.De tous les ordres [^1769], celui des chefs de famille (est le premier), il n'y a aucun doute là-dessus. L'inaperçu est la source des mondes ; et le même est aussi la fin de toute chose. Les jours finissent avec le coucher (du soleil) [^1770] ; la nuit finit avec le lever (du soleil) ; la fin du plaisir est toujours le chagrin ; <span id="p355"><sup><small>[ p. 355 ]</small></sup></span> la fin du chagrin est toujours le plaisir. Toutes les accumulations finissent par l'épuisement ; toutes les ascensions finissent par des chutes ; toutes les associations finissent par des dissociations ; et la vie finit par la mort. Toute action finit par la destruction ; la mort est certaine pour tout ce qui est né [1](Anugita_30#fn1771) ; (tout) mobile ou immobile dans ce monde est toujours transitoire. Sacrifice, don, pénitence, étude, observances et règles, tout cela aboutit à la destruction [2](Anugita_30#fn1772). La connaissance est sans fin. C'est pourquoi celui dont le moi est tranquille, dont les sens sont soumis, qui est dépourvu de l'idée que ceci ou cela m'appartient, qui est exempt d'égoïsme, est libéré de tous les péchés par la pure connaissance.

## Notes de bas de page

[^1756]: 352:1 Voir [p. 324](Anugita_22#p324) supra.

[^1757]: 352:2 Voir Gîtâ, [p. 101](Bhagavadgita_12#p101). Immobile signifie probablement « non perturbé par les qualités » (Gîtâ, [p. 110](Bhagavadgita_14#p110)), ou peut-être la même chose que « d'esprit stable » dans Gîtâ, [p. 101](Bhagavadgita_12#p101). Le sens est à peu près le même aux deux endroits.

[^1758]: 352:3 Qui a naissance &c., Nîlaka<i>nth</i>a, c'est-à-dire toute la création, je présume.

[^1759]: 352:4 Les noms, c'est-à-dire, des divers éléments, et leurs qualités.

[^1760]: 352:5 Cela doit vouloir dire deux semaines.

[^1761]: 352:6 Ceci est spécifié, dit Ar<i>g</i>una Mi<i>s</i>ra, car les six mois du solstice d'hiver sont causés par la présence du soleil à ce Nakshatra. Quant à ces six mois, cf. Gîtâ, [p. 81](Bhagavadgita_8#p81). Pour la même raison, ajoute Ar<i>g</i>una Mi<i>s</i>ra, la saison hivernale est mentionnée comme la meilleure.

[^1762]: 352:7 Le mot âdi, littéralement commencement, est utilisé dans tout ce passage dans différents sens ; il signifie la source, il signifie le meilleur, et il signifie le premier dans l'ordre.

[^1763]: 353:1 Ceci doit être comparé avec l'énumération à [p. 345](Anugita_27#p345) supra, et celle de la Gîtâ à laquelle il est fait référence.

[^1764]: 353:2 Cf. [p. 346](Anugita_28#p346) supra. Nîlaka<i>nth</i>a prend le feu pour signifier le feu gastrique, et bhûta, rendu par nous éléments, pour signifier l'espèce d'êtres nés d'œufs et d'utérus.

[^1765]: 353:3 Le célèbre verset « Tat savitur », etc. Voir entre autres Brihaðaranyaka, p. 999; Actes I, 1, 1, 9; Manu II, 77 seq., 104-170.

[^1766]: 353:4 Voir pp. [264](Anugita_6#p264), [265](Anugita_6#p265) supra.

Il n'abandonne pas entièrement le fil de son discours, mais insère simplement cette petite clause. Nîlaka<i>nth</i>a ajoute : Sâvitrî inclut ici tous les modes d'adoration prescrits pour les Brâhma<i>n</i>as, etc., et même pour les Mle<i>nth</i>as. Cf. note 3, et Gautama (éd. Bühler), p. 74 note.

[^1768]: 353:6 Cf. <i>Khândogya</i>, p. 109, et le commentaire de <i>Sankara</i>. Ar<i>g</i>una Mi<i>s</i>ra compare ce texte : Tasmâdesha eteshâm pa<i>s</i>unâm sresh<i>th</i>atamogab. Je ne sais pas où il apparaît.

[^1769]: 353:7 Cf. Santi Parvan (Râgadharma), chap. II, art. 11.

[^1770]: 353:8 C'est-à-dire Vâsuki, Nîlaka<i>nth</i>a. Il s'agit plus probablement de l'espèce.

[^1771]: 353:9 Comme il est utilisé dans diverses cérémonies.

[^1772]: 354:1 C'est-à-dire le Créateur ; son champ signifie, je présume son siège spécial.

[^1776]: 354:5 À savoir Brahma<i>k</i>ârin, chef de famille, forestier et Samnyâsin. Cf. <i>S</i>ânti Parvan (Moksha), ch. 191, st. 10 ; Manu VI, 89 ; Gautama, p. 190.
