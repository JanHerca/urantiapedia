## Nouvelles langues

Urantiapedia est désormais disponible en [croate](/hr/home) ! :tada: :clap: Cela porte à 28 le nombre de langues prises en charge. Dans cette langue, comme pour la plupart des autres, nous avons inclus le Livre d'Urantia, la Bible, la Paramonie (ou les références croisées entre le Livre d'Urantia et la Bible), les pages d'accueil et les informations juridiques. Urantiapedia contient désormais toutes les langues disposant d'au moins une traduction du Livre d'Urantia.

## :blue_book: Le Livre d'Urantia

Nous avons terminé la mise à jour du Livre d'Urantia dans toutes les langues, en corrigeant certaines erreurs et en augmentant le nombre de traductions :
- Tous les documents du Livre d'Urantia sont maintenant disponibles dans toutes les langues (28 actuellement).
- Toutes les langues disposent enfin de la version multi-colonnes, avec la version anglaise originale et la version traduite.
- Une version multi-colonnes a également été ajoutée à la version anglaise (toutes les langues en disposent donc désormais). La version anglaise multi-colonnes affiche à la fois la version actuelle et la version originale de 1955. La version actuelle inclut quelques modifications mineures et des corrections d'errata.
- Dans les versions coréenne, russe, française et espagnole, nous avons ajouté de nouvelles traductions alternatives. Il existe quatre traductions en coréen, deux en russe, deux en français et quatre en espagnol.
- Nous avons corrigé certains paragraphes spéciaux du *Livre d'Urantia* qui servent simplement de séparateurs de paragraphes et sont marqués d'astérisques (* * *). Ces paragraphes sont soit ceux-ci, soit ils se trouvent après ceux-ci : [LU 31:10.21](/fr/The_Urantia_Book/31#p10_21), [LU 56:10.22](/fr/The_Urantia_Book/56#p10_22), [LU 120:3.11](/fr/The_Urantia_Book/120#p3_11), [LU 134:6.14](/fr/The_Urantia_Book/134#p6_14), [LU 144:5.11-87](/fr/The_Urantia_Book/144#p5_11). Ces paragraphes ont désormais les mêmes références que celles des publications officielles de la [Fondation Urantia](https://urantia.org).
- Nous avons décidé que la section 139:9 (Jacques et Jude, Alphaeus) n'existe pas et que la section existante est [LU 139:10](/fr/The_Urantia_Book/139#p10) afin d'unifier l'accès à cette section atypique.
- Nous avons inclus les noms des auteurs des documents dans l'index compact.
- Nous avons ajouté des éléments à l'index étendu, dans les sections qui figurent déjà dans l'index officiel. Ces ajouts concernent uniquement l'anglais et l'espagnol. Pour les autres langues, une traduction a été effectuée à partir de l'anglais.
- La typographie en coréen a fait l'objet de quelques ajustements.


## :books: La Bibliothèque

Dans la [Bibliothèque](/fr/index/books), nous avons commencé le traitement des livres bibliques restants à télécharger (les livres apocryphes) dans presque toutes les langues. Actuellement, ils ne sont disponibles qu'en anglais, en espagnol et en français. L'objectif est de rendre ces livres disponibles dans la Bibliothèque dans toutes les langues. Ce processus nécessitera une mise à jour des logiciels _Urantiapedia Tools_ et _Wikijs_ afin de permettre la création et le téléchargement de ces fichiers plus rapidement. Nous travaillons actuellement sur une nouvelle version d'_Urantiapedia Tools_.

## :wrench: Urantiapedia Tools

Nous avons commencé le développement d'une nouvelle version d'_Urantiapedia Tools_. Ce logiciel permet d'accélérer de nombreux processus automatisés. Jusqu'à présent, le code source de ce logiciel se trouvait dans le dossier [`app`](https://github.com/JanHerca/urantiapedia/tree/master/app) du projet. Ce code y restera pour consultation ultérieure, mais la version 2.0 est développée en tant que projet distinct sur GitHub : [Urantiapedia Tools 2.0](https://github.com/JanHerca/urantiapedia-tools). Ce nouveau projet est public, comme tout ce que nous faisons chez Urantiapedia ; toute personne souhaitant y contribuer est donc la bienvenue. L'objectif est de porter tout le code utile restant, de supprimer le code obsolète, d'améliorer la maintenabilité du logiciel (grâce à l'utilisation des bibliothèques les plus récentes et modernes) et d'optimiser considérablement les processus de traduction afin d'accélérer significativement la traduction de milliers de pages.

En parallèle, nous allons commencer le développement d'une nouvelle version de Wikijs. Ce logiciel alimente le site web Urantiapedia. Il s'agit d'un logiciel libre, et nous en avons créé une copie ici : [Wikijs pour Urantiapedia](https://github.com/JanHerca/wiki). Nous souhaitons y apporter de nombreuses améliorations et les proposer ensuite au projet original sous forme de contributions. Notre principal défi est la migration du contenu d'Urantipedia, actuellement disponible en trois langues, vers un contenu disponible dans près de 30 langues. Cela implique de multiplier par dix le nombre de pages web. Urantiapedia compte actuellement environ 120 000 pages. Le chargement des pages est un processus lent qui s'exécute en arrière-plan afin de ne pas impacter l'ergonomie du site. Nous souhaitons mettre en place des systèmes qui accélèrent ces chargements afin de pouvoir héberger les plusieurs millions de pages nécessaires pour finaliser Urantiapedia dans toutes les langues d'ici quelques mois. La nouvelle version de Wikijs que nous développons est également un projet public ouvert aux collaborations.
