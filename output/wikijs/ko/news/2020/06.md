## :sparkles:새로운 콘텐츠

- 웹사이트는 아직 이용할 수 없습니다. 콘텐츠 부분에서는 계속해서 [GitHub](https://github.com/JanHerca/urantiapedia)에 직접 콘텐츠를 업로드하고 있습니다.
- _우란시아서_ 파일은 JSON과 위키텍스트로 생성되었습니다. 이 두 가지 형식에 대한 자세한 내용은 아래를 참조하세요.
- 우리는 _성경_과 _우란시아 책_ 사이의 교차 참조를 계속 포함하고 있으며, [파라모니](https://urantia-book.org/urantiabook/paramony/)는 이 책의 독자인 듀안 포가 만든 기존 목록입니다.
- 현재 모든 콘텐츠는 스페인어로만 제공됩니다.

## :wrench: Urantiapedia 도구

- _Urantiapedia Tools_의 복잡성을 고려하여 C#에서 Javascript로 코드를 마이그레이션하고 [Electron](https://www.electronjs.org/)을 데스크톱 실행 환경으로 사용하기로 결정했습니다. 매우 빠른 코딩 언어인 Javascript를 사용하면 모든 개발 과정을 간소화할 수 있을 뿐만 아니라, 웹 호환 코드이므로 향후 웹 페이지 자체에서 재사용할 수도 있습니다.
- _우란시아 서_에서 중간 [JSON](https://en.wikipedia.org/wiki/JSON) 형식으로 [LaTeX](https://en.wikipedia.org/wiki/LaTeX) 형식 변환기를 추가했습니다. 이 형식은 교차 참조를 추가하는 작업을 더 편안하게 처리할 수 있게 해줍니다.
- _유란시아 책_을 JSON 형식에서 위키텍스트 형식으로 변환하는 기능이 추가되었습니다. 이 형식은 Wikipedia 소프트웨어인 [Mediawiki](https://www.mediawiki.org/wiki/MediaWiki)에서 사용되는 형식이며, 우리는 Urantiapedia에서도 사용할 것입니다. 이 변환기는 또한 프로세스에 Paramony 각주를 추가합니다.
