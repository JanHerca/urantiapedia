이 HTML 편집기는 HTML 지식이 있는 기여자를 위한 것입니다. 미리보기 기능은 제공하지 않으며, 주로 페이지의 내부 HTML 코드를 확인하는 데 중점을 두고 있습니다. 어떤 편집기를 사용하든 모든 페이지는 최종적으로 HTML 코드를 생성하기 때문입니다.

이 편집기가 다른 편집기보다 좋은 점은 사용법을 잘 안다면 다른 편집기보다 모든 유형의 콘텐츠를 삽입할 수 있고, `클래스 이름`을 통해 모든 유형의 스타일을 적용할 수 있으며, 다른 편집기에서는 사용할 수 없는 요소도 추가할 수 있다는 것입니다.

> 마크다운 편집기는 HTML을 사용하는 모든 기능을 제공한다는 점에 유의하세요. 마크다운은 같은 페이지에서 마크다운과 HTML 마크업을 모두 지원하는 형식이기 때문입니다.
{.is-info}

스크린샷은 HTML 편집기의 가장 눈에 띄는 컨트롤을 보여줍니다.
1. 페이지 제목
2. 버튼 바 저장:
    - **저장됨/저장** 버튼. **저장됨**이라고 표시되면 변경 사항이 없는 것입니다. **저장**이라고 표시되면 변경 사항을 영구적으로 적용하려면 클릭하세요. **페이지는 자동 저장되지 않으므로 자주 저장하세요!** :point_left:
    - **페이지** 버튼: 제목, 설명, 태그 등 페이지의 메타데이터를 다시 수정할 수 있습니다.
    - **닫기** 버튼: 페이지를 읽고 나면 여기를 닫아 읽기 모드로 돌아갑니다.
3. 서식 버튼 막대. 텍스트를 선택하고 클릭하여 적용하거나 커서를 삽입할 위치를 지정합니다. 옵션은 다음과 같습니다(왼쪽에서 오른쪽 순).
    - 링크 삽입(*아직 사용할 수 없음*, 수동으로 삽입해야 함)
    - 이미지 삽입
    - 블록(*아직 사용할 수 없음*, 수동으로 삽입해야 함)
    - 코드(*아직 사용 불가*, 수동으로 입력해야 함)
    - 다이어그램(*아직 사용 불가*, 수동으로 삽입해야 함)
    - 수학 표현식(*아직 사용 불가*, 수동으로 삽입해야 함)
