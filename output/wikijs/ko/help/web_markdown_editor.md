스크린샷은 Markdown 편집기의 가장 중요한 컨트롤을 보여줍니다.
1. 페이지 제목
2. 버튼 바 저장:
    - **저장됨/저장** 버튼. **저장됨**이라고 표시되면 변경 사항이 없는 것입니다. **저장**이라고 표시되면 변경 사항을 영구적으로 적용하려면 클릭하세요. **페이지는 자동 저장되지 않으므로 자주 저장하는 것을 잊지 마세요!** :point_left:
    - **페이지** 버튼: 제목, 설명, 태그 등 페이지의 메타데이터를 다시 수정할 수 있습니다.
    - **닫기** 버튼: 페이지를 읽고 나면 여기를 닫아 읽기 모드로 돌아갑니다.
3. 서식 버튼 막대. 텍스트를 선택하고 적용을 클릭하세요. 옵션은 다음과 같습니다(왼쪽에서 오른쪽 순).
    - 용감한
    - 이탤릭체
    - 취소선
    - 헤더
    - 아래첨자
    - 상위 첨자
    - 인용문과 블록
    - 순서 없는 목록
    - 순서가 있는 목록
    - 코드 블록
    - 키보드 스타일
    - 수평선
4. 버튼 막대를 삽입합니다. 삽입할 위치에 커서를 놓고 클릭합니다. 옵션은 다음과 같습니다(위에서 아래로):
    - Urantiapedia 내의 다른 페이지로 연결합니다. (마크다운으로 다른 웹사이트로 연결하는 경우, URL만 입력하면 됩니다.)
    - 이미지: 이미지 선택기를 엽니다.
    - 블록 (*아직 이용 불가*)
    - 코드(*아직 제공되지 않음*)
    - 비디오 및 오디오(*아직 제공되지 않음*)
    - 다이어그램. `Draw.io` 편집기를 열고 이를 통해 만든 다이어그램을 삽입할 수 있습니다.
    - 수학 표현식(*아직 제공되지 않음*)
    - 표에 대한 도움말 (*아직 제공되지 않음*)
5. 마크다운 서식 도움말: 이 버튼을 클릭하면 마크다운 서식을 작성하는 방법에 대한 예 목록이 열립니다.
