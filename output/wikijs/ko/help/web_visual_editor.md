이 시각적 편집기는 인기 있는 [CKEditor 5](https://ckeditor.com/)를 기반으로 한 WYSIWYG(What You See Is What You Get) 편집기입니다.

시각적 편집기의 가장 중요한 컨트롤은 스크린샷에 표시되어 있습니다.
1. 페이지 제목
2. 버튼 바 저장:
    - **저장됨/저장** 버튼. **저장됨**이라고 표시되면 변경 사항이 없는 것입니다. **저장**이라고 표시되면 변경 사항을 영구적으로 적용하려면 클릭하세요. **페이지는 자동 저장되지 않으므로 자주 저장하세요!** :point_left:
    - **페이지** 버튼: 제목, 설명, 태그 등 페이지의 메타데이터를 다시 수정할 수 있습니다.
    - **닫기** 버튼: 페이지를 읽고 나면 여기를 닫아 읽기 모드로 돌아갑니다.
3. 서식 버튼 막대. 텍스트를 선택하고 클릭하여 적용하거나 커서를 삽입할 위치를 지정합니다. 옵션은 다음과 같습니다(왼쪽에서 오른쪽 순).
    - 헤더
    - 글꼴 크기
    - 글꼴 패밀리
    - 굵게 (<kbd>CTRL</kbd> + <kbd>B</kbd>)
    - 이탤릭체(<kbd>CTRL</kbd> + <kbd>I</kbd>)
    - 밑줄(<kbd>CTRL</kbd> + <kbd>U</kbd>)
    - 취소선(<kbd>CTRL</kbd> + <kbd>SHIFT</kbd> + <kbd>X</kbd>)
    - 아래첨자
    - 상위 첨자
    - 색상으로 텍스트 표시 / 명확한 마크업
    - 텍스트 정렬
    - 번호가 매겨진 목록
    - 번호가 없는 목록
    - 체크박스가 있는 할 일 목록
    - 특수문자 삽입
    - 마법사를 사용하여 다른 Urantiapedia 페이지에 대한 링크 삽입
    - 모든 페이지에 링크 삽입 (<kbd>CTRL</kbd> + <kbd>L</kbd>)
    - 인용문
    - 이미지 삽입
    - 표 삽입
    - 코드 스타일
    - 코드 블록
    - 비디오 삽입
    - 수평선
    - 서식 제거(*아직 사용할 수 없음*)
    - 실행 취소(<kbd>CTRL</kbd> + <kbd>Z</kbd>)
    - 다시 실행(<kbd>CTRL</kbd> + <kbd>Y</kbd>)


## 향후 기능

편집자의 몇 가지 중요한 동작은 아직 구현되지 않았지만 가까운 시일 내에 구현될 예정입니다.
- 각주 삽입
- *우란시아서* 또는 *성경*에 대한 참조에 대한 자동 링크.
- 인용문 스타일의 다양한 블록 유형.
- 수학적 표현.
- `Draw.io` 편집기를 사용한 다이어그램.
