Urantiapedia에 콘텐츠를 커밋하기 위해 [Urantiapedia](https://urantiapedia.org) 웹사이트의 전체 백업이 포함된 GitHub 프로젝트가 사용됩니다. 프로젝트 위치는 다음과 같습니다.

[우란티아페디아 백업](https://github.com/JanHerca/urantiapedia-backup)

이 프로젝트는 웹사이트와 동기화되어 있으므로 여기에서 *우란티페디아*의 모든 내용의 전체 사본을 검색할 수 있습니다.

콘텐츠를 커밋하려면 다음 단계를 따르세요.
- GitHub에서 *Urantiapedia-backup* 프로젝트를 복제하거나 포크합니다.
- *Urantiapedia* 프로젝트의 `output/wikijs` 폴더에 있는 새 콘텐츠를 *Urantiapedia-backup* 프로젝트의 루트 폴더로 복사합니다.
- 풀 리퀘스트를 실시합니다.
- GitHub 프로젝트의 *관리자* 사용자는 변경 사항을 병합합니다. 변경 사항이 병합되면 업로드 작업을 실행하는 서버의 현재 성능에 따라 몇 시간 내에 *Urantiapedia* 웹 페이지에 반영됩니다.
