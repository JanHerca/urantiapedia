لنشر محتوى على يورانتيابيديا، يُستخدم مشروع على GitHub مع نسخة احتياطية كاملة لموقع يورانتيابيديا (https://urantiapedia.org). المشروع موجود هنا:

[نسخة احتياطية من Urantiapedia](https://github.com/JanHerca/urantiapedia-backup)

تم مزامنة هذا المشروع مع الموقع الإلكتروني حتى يمكن استرجاع نسخة كاملة من كل محتوى *Urantiapedia* منه.

للالتزام بالمحتوى اتبع الخطوات التالية:
- استنساخ أو شوكة مشروع *Urantiapedia-backup* في GitHub.
- انسخ المحتوى الجديد في مجلد `output/wikijs` الخاص بمشروع *Urantiapedia* إلى المجلد الجذر في مشروع *Urantiapedia-backup*.
- قم بعمل طلب سحب.
- سيقوم مستخدمو مشروع GitHub *المسؤولون* بدمج التغييرات. بمجرد دمجها، ستكون متاحة خلال الساعات القادمة على صفحة *Urantiapedia*، وذلك حسب الأداء الحالي للخوادم لتنفيذ مهام التحميل.
