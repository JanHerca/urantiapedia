为了向 Urantiapedia 提交内容，我们使用了一个 GitHub 项目，其中包含 [Urantiapedia](https://urantiapedia.org) 网站的完整备份。该项目位于：

[Urantiapedia-backup](https://github.com/JanHerca/urantiapedia-backup)

该项目与网站同步，因此可以从中检索 *Urantiapedia* 所有内容的完整副本。

要提交内容，请按照以下步骤操作：
- 在 GitHub 中克隆或分叉 *Urantiapedia-backup* 项目。
- 将 *Urantiapedia* 项目的 `output/wikijs` 文件夹中的新内容复制到 *Urantiapedia-backup* 项目的根文件夹中。
- 发出拉取请求。
- GitHub 项目的 *管理员* 用户将合并更改。更改合并后，将在接下来的几个小时内在 *Urantiapedia* 网页上提供，具体取决于服务器当前执行上传作业的性能。
