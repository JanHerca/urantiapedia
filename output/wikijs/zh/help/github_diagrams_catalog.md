以下是如何创建《尤兰提亚之书》图表目录。

## 过程

该流程与创建文章的流程相同。请参阅[创建文章目录](/zh/help/github_articles_catalog)。图表的临时列表可在此处找到：[计划中的图表](/en/index/diagrams)。

要编辑该列表，您可以通过两种方式进行：

- 在此处登录 _Urantiapedia_。前往图表列表：[计划图表](/en/index/diagrams)。在右上角按钮面板的 _页面操作 > 编辑_ 菜单中，将页面置于编辑状态。一个简单的 Markdown 编辑器将打开，您可以在其中编辑列表。（重要提示：要执行此操作，您需要管理员权限来启用该页面的编辑权限。）
- 在 GitHub 项目中修改此文件：`/output/wikijs/en/index/diagrams.md`。要修改它，您可以直接下载该文件或从 GitHub 下载整个项目。有关如何从 GitHub 下载内容，请参阅[助理编辑帮助](/zh/help/github_assistant)。

在前两种情况下，要编辑的文件是一个非常简单的 Markdown 文件，其中每个图表都必须以表格的形式包含以下内容：

- `:white_square_button:` 标志表示图表尚未创建。
- 标题。
——作者。
- 一些对图表进行分类的可选标签。
- 可下载该图表的 URL。

例如：

```
| Status                | Title                              | Author  | Tags  | URL                               |
| ----------------------| -----------------------------------| --------| ----- | --------------------------------- |
| :white_square_button: | "The Kingdom of Heaven Revelation" |         |       | http://www.universalfather.info/  |
```

## 发送文件

如果您已经从 GitHub 上的文件修改了列表，则有两种方式提交更改：如果您是“主编”，请按照[主编帮助](/zh/help/github)中的详细说明使用 GitHub；如果您是“助理编辑”，您可以直接将修改后的 Markdown 文件发送到任何管理员的电子邮箱或 urantiapedia@gmail.com。
