屏幕截图显示了 Markdown 编辑器的最重要控件：
1. 页面标题
2. 保存按钮栏：
    - **已保存/保存**按钮。如果显示**已保存**，表示没有更改；如果显示**保存**，请点击使更改永久生效。**页面不会自动保存，所以别忘了经常保存！** :point_left:
    - 按钮**页面**：允许您重新修改页面的元数据，例如标题、描述和标签。
    - **关闭**按钮：当您读完该页面后，关闭此处返回阅读模式。
3. 格式按钮栏。选择文本并点击即可应用。选项如下（从左到右）：
    - 大胆的
    - 斜体
    - 删除线
    - 标题
    - 下标
    - 上标
    - 引文和区块
    - 无序列表
    - 有序列表
    - 代码块
    - 键盘风格
    - 水平线
4. 插入按钮栏。将光标放在要插入的位置并点击。选项如下（从上到下）：
    - 链接到 Urantiapedia 内的其他页面。（在 Markdown 中链接到其他网站，只需输入 URL，无需更多内容）。
    - 图像：打开图像选择器。
    - 块（*尚不可用*）
    - 代码（*尚不可用*）
    - 视频和音频（*尚不可用*）
    - 图表。打开`Draw.io`编辑器并允许您插入用它创建的图表。
    - 数学表达式（*尚不可用*）
    - 表格帮助（*尚不可用*）
5. Markdown 格式帮助：此按钮打开如何编写 Markdown 格式的示例列表。
