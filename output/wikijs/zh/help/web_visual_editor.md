此可视化编辑器是一个所见即所得 (WYSIWYG) 编辑器，基于流行的 [CKEditor 5](https://ckeditor.com/)。

可视化编辑器最重要的控件如屏幕截图所示：
1. 页面标题
2. 保存按钮栏：
    - **已保存/保存**按钮。如果显示**已保存**，表示没有更改；如果显示**保存**，请点击使更改永久生效。**页面不会自动保存，所以别忘了经常保存！** :point_left:
    - 按钮**页面**：允许您重新修改页面的元数据，例如标题、描述和标签。
    - **关闭**按钮：当您读完该页面后，关闭此处返回阅读模式。
3. 格式按钮栏。选择文本并点击以应用或定位光标进行插入。选项如下（从左到右）：
    - 标题
    - 字体大小
    - 字体系列
    - 粗体（<kbd>CTRL</kbd> + <kbd>B</kbd>）
    - 斜体（<kbd>CTRL</kbd> + <kbd>I</kbd>）
    - 下划线（<kbd>CTRL</kbd> + <kbd>U</kbd>）
    - 删除线 (<kbd>CTRL</kbd> + <kbd>SHIFT</kbd> + <kbd>X</kbd>)
    - 下标
    - 上标
    - 用颜色标记文本/清除标记
    - 文本对齐
    - 编号列表
    - 未编号列表
    - 带有复选框的待办事项列表
    - 插入特殊字符
    - 使用向导插入另一个 Urantiapedia 页面的链接
    - 插入任意页面的链接 (<kbd>CTRL</kbd> + <kbd>L</kbd>)
    - 引号
    - 插入图片
    - 插入表格
    - 代码风格
    - 代码块
    - 插入视频
    - 水平线
    - 删除格式（*尚不可用*）
    - 撤消 (<kbd>CTRL</kbd> + <kbd>Z</kbd>)
    - 重做 (<kbd>CTRL</kbd> + <kbd>Y</kbd>)


## 未来功能

编辑器的一些重要操作尚未实现，将在不久的将来完成：
- 插入脚注
- 自动链接到《尤兰提亚之书》或《圣经》。
- 不同类型的引文风格的块。
- 数学表达式。
- 使用`Draw.io`编辑器绘制图表。
