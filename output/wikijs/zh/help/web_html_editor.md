此 HTML 编辑器面向具备 HTML 知识的贡献者。它不提供预览功能，主要目的是让您能够查看页面的内部 HTML 代码，因为所有页面，无论使用什么编辑器生成，最终都会生成 HTML 代码。

如果您知道如何很好地使用它，那么这个编辑器比其他编辑器的优势在于，它允许您插入任何其他编辑器所没有的任何类型的内容，通过`classnames`应用任何类型的样式，以及添加其他编辑器中不可用的元素。

> 需要注意的是，Markdown 编辑器也允许使用 HTML 的所有功能，因为 Markdown 是一种支持在同一页面中混合使用 Markdown 和 HTML 标记的格式。
{.is-info}

屏幕截图显示了 HTML 编辑器最突出的控件：
1. 页面标题
2. 保存按钮栏：
    - **已保存/保存**按钮。如果显示**已保存**，表示没有更改；如果显示**保存**，请点击使更改永久生效。**页面不会自动保存，所以别忘了经常保存！** :point_left:
    - 按钮**页面**：允许您重新修改页面的元数据，例如标题、描述和标签。
    - **关闭**按钮：当您读完该页面后，关闭此处返回阅读模式。
3. 格式按钮栏。选择文本并点击以应用或定位光标进行插入。选项如下（从左到右）：
    - 插入链接（*尚不可用*，必须手动插入）
    - 插入图片
    - 块（*尚不可用*，必须手动插入）
    - 代码（*尚不可用*，必须手动插入）
    - 图表（*尚不可用*，必须手动插入）
    - 数学表达式（*尚不可用*，必须手动插入）
