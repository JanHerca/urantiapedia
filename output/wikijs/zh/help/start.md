## 什么是？

[Urantiapedia](https://urantiapedia.org) 是一个用于发布、分享和协作任何与 [《尤兰提亚之书》](https://www.urantia.org/) 相关内容的网站。


## 我们的使命

我们为《Urantiapedia》设定的关键任务是：
1. 帮助在全世界传播《尤兰提亚之书》的启示。
2. 让人们更好地理解《尤兰提亚之书》的教义。
3. 提供与《尤兰提亚启示》相关的大部分现有知识的链接，以提供实用性。


## 它是如何工作的？

它基于 [Wiki.js](https://js.wiki/)，这是一款功能强大且可扩展的开源 Wiki 软件，它基于 [Node.js](https://nodejs.org/) 和 [Vue.js](https://vuejs.org/) 等现代技术构建。任何组织都可以自由使用 *Wiki.js* 来创建任何类型的 Wiki。

Wiki 是由用户社区协作维护和编辑的网页。任何注册用户在经过验证后，都可以添加、修改或删除内容，所有更改都会永久保存在历史记录中，即使意外删除也不会造成任何损失。信息存储在数据库中，因此可以执行诸如大规模添加/删除、备份或搜索等常规操作。

为了方便添加内容，一个名为“Urantiapedia Tools”的程序正在开发中。该程序可以自动执行格式转换任务，并协助内容审核工作。

随着 *Urantiapedia* 扩展其内容类型，*Urantiapedia Tools* 和 *Wiki.js* 都将获得新功能。开发内容和内容文件可从 [Urantiapedia at GitHub](https://github.com/JanHerca/urantiapedia) 获取。该网站是公开的，任何人都可以按照 GitHub 的方法或通过电子邮件发送内容进行贡献。

## 它是如何组织的？

与任何 Wiki 一样，内容以页面形式呈现，每个页面都有唯一的 URL 和标题。以下是预期的主要内容：

* **《尤兰提亚之书》简报档案** - 收录读者在协会和个人出版物上撰写的文章。每篇文章单独发布一页。[阅读更多](/en/article)。
* **尤兰提亚图书馆** - 馆藏书籍包括《尤兰提亚之书》、《圣经》、世界各主要宗教的书籍以及其他与《尤兰提亚启示录》相关的文本。该图书馆还将收录希望公开出版作品的读者所著书籍。每本书的每一章都属于《尤兰提亚百科全书》的一页。[阅读更多](/en/book)。
* **《尤兰提亚之书》百科全书** - 是一部涵盖《尤兰提亚之书》中所有相关主题或专有名词的百科全书。每个主题都存储在一页纸上。本网站有时将其称为“主题索引”，但它远不止是一个索引。[阅读更多](/en/topic)。
* **《尤兰提亚之书》地图库**：将包含二维或三维互动地图，展示《尤兰提亚启示录》中描述的地点或旅程；此外，还将提供非互动地图的可下载文件。每张地图占一页。地图可以是静态的，也可以包含动画，用于创建动态路线。[阅读更多](/en/map)。
* **《尤兰提亚之书》媒体库**：将成为各种视听材料的存储库，包括以下内容：
  * 图像，可以是说明《尤兰提亚之书》概念的图画或照片。
  * 图表或示意图，如果它们太大，将尝试使其具有交互性，以便在小型设备上查看。
  * 幻灯片。
  * 3D 模型，每个模型占一页，可以是静态的，也可以是交互式的，如地图一样。
  *《尤兰提亚之书》的音频版本。
  * 网站上嵌入的视频集，如果不可能，则提供视频链接，其中包含有关《尤兰提亚之书》的课程或解释。

有关内容的更多信息，请查看此处：[内容](/zh/help/content)


## 项目里程碑

对于要收录到 *Urantipedia* 中的每种语言，所有任务都将按照特定的里程碑进行：
1. 里程碑一——《尤兰提亚之书》和《圣经》。
2. 里程碑 II——《报纸图书馆》（文章、学习辅助工具）、《尤兰提亚图书馆》（书籍）和《尤兰提亚书籍百科全书》。
3. 里程碑三——翻译器。这并非内容本身，而是《Urantiapedia 工具》中的一个应用程序，用于快速翻译成各种语言。
4. 里程碑 IV——对里程碑 I 和 II 的内容进行扩展和修正，并添加_地图库_（地图）和_媒体库_（图表、演示文稿、音频、视频、3D 模型）。

要了解项目的里程碑，请查看[项目里程碑](/zh/help/phases)。

## 当前状态

目前，*Urantiapedia* 的内容如下：
* 里程碑 I 已完成 28 种语言的翻译。《尤兰提亚之书》、《圣经》以及《帕拉蒙尼》（或两者之间的联系）均已添加所有这些语言的版本。
* 里程碑 II 和 III 已完成三种语言的翻译：英语、西班牙语和法语。《报纸图书馆》、《尤兰提亚图书馆》和《百科全书》中的内容足以证明这些里程碑的实现。
* 里程碑 IV 中仅完成了小规模的概念验证。目前我们正处于这个阶段。

要了解有关项目当前进度的更多信息，请查看此处：[状态](/zh/help/status)。


## 如何合作

Urantiapedia 并非一个向所有人开放注册的网站。我们的运营方式与维基百科不同。原因是 Urantiapedia 过去曾有一个版本允许开放注册，并导致了严重的垃圾邮件问题。如果没有完善的安全系统和优秀的协作团队来防范这些入侵，开放注册系统的问题将难以避免。因此，如果有人希望在管理或编辑层面进行协作，则必须申请注册。但是，如果仅希望进行评论方面的协作，则无需在 Urantiapedia 上注册。

因此，如果您想合作这个项目，最好的联系方式是：
* **通过电子邮件**。请发送电子邮件至 <a href="mailto:urantiapedia@gmail.com">urantiapedia@gmail.com</a>。如果您希望在 Urantiapedia 注册为 *管理员*、*主编*、*作者* 或 *开发者*，请发送电子邮件提出您的申请。如果您希望以 *助理编辑* 或 *审稿人* 的身份做出贡献，请将您的建议或更正发送至同一邮箱。我们建议您将建议或更正作为邮件正文的一部分发送，而不是以 Word、PDF 或类似文件的形式发送。
* **通过 [GitHub](https://github.com/JanHerca/urantiapedia)**。如本文档所述，整个项目在 GitHub 上公开开放。任何人都可以注册 GitHub，创建项目的 *fork*，添加所需的更改，然后创建 *pull request*（典型的 GitHub 更改请求）。提交的更改将由 Urantiapedia 的 *管理员* 进行评估，并可能根据需要接受或拒绝。请参阅帮助的其余部分，了解更多关于此协作模式的信息。
* **通过网站 [Blue Fields](https://blue-fields.netlify.app/)**，待该网站完成并投入运营后即可使用。该网站是一个与《尤兰提亚百科全书》无关的项目，旨在团结和组织那些希望参与世界各地智慧、有目的且利他项目的《尤兰提亚之书》读者。在该网站注册后，项目团队将与您联系。

*Urantiapedia* 合作者的不同角色如下：
* *管理员*：负责维护网络服务器、页面、批量数据上传，并且是唯一有权编辑《尤兰提亚之书》、《圣经》和其他特定内容的人员。您必须发送电子邮件至 urantiapedia@gmail.com，并在通过相应审核后获得*管理员*的权限。这些用户最终将在 GitHub 和 Urantiapedia 上分别拥有一个帐户。了解更多信息，请访问：[管理员手册](/zh/help/admin)。
* *主编*：负责通过本网站和 GitHub 项目进行修改和翻译贡献。您必须发送电子邮件至 urantiapedia@gmail.com，并在通过相应审核后方可获准成为*主编*。这些用户最终将在 GitHub 和 Urantiapedia 分别拥有一个帐户。请参阅：[主编手册](/zh/help/github)
* *作者*：这些是*主编*，他们除了可以编辑任何*主编*可以编辑的内容外，也是唯一有权编辑其所撰写内容的编辑者。需要发送电子邮件至 urantiapedia@gmail.com，通过相应验证后即可成为*作者*。这些用户最终将在 GitHub 和 Urantiapedia 上分别拥有一个帐户。
* *助理编辑*：他们只能使用 GitHub 发送修改请求。他们没有本网站的账户，因此无需提交任何请求或通过任何验证。这些用户最终会在 GitHub 上拥有一个账户。请参阅：[助理编辑手册](/zh/help/github_assistant)
* *审核员*：他们的任务是确保网站上的内容正确无误，检测上传内容时可能出现的印刷、翻译或编写错误。
* *开发者*：他们在本网站和 GitHub 上拥有账户，并负责开发 *Urantiapedia 工具* 和 *Wiki.js* 的功能。需要发送电子邮件至 urantiapedia@gmail.com，通过相应验证后即可成为 *开发者*。这些用户最终将在 GitHub 和 Urantiapedia 上分别拥有一个账户。请参阅：[开发者手册](/zh/help/devs)。
* *大使*：他们无需在本网站或 GitHub 上拥有账户。他们的目的是传播 *Urantiapedia* 项目，并让各协会和公众知晓。
* *视频创作者*：负责为 [Urantiapedia YouTube 频道](https://bit.ly/3zOuxvB) 创作素材。您需要发送电子邮件至 urantiapedia@gmail.com，通过相应验证后即可成为*视频创作者*。我们将提供 Urantiapedia YouTube 帐户的访问权限。该频道上传有关 Urantiapedia 项目、合作方式以及如何最大程度利用网站资源的讲解视频。

要了解有关我们希望合作的个人资料的更多信息，请查看[合作者个人资料]（/zh/help/roles）。
