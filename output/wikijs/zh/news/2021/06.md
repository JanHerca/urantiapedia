<img src="/_assets/svg/icon-news.svg" alt="新闻" style="width: 80px;"> 本月，我们将继续推进《尤兰提亚百科全书》的多语言化工作。我们已将《尤兰提亚之书》的所有脚注合并到一个 JSON 文件中。这将使我们能够快速将这些脚注翻译成其他语言。:globe_with_meridians:

## :sparkles:新内容

- 在 [GitHub](https://github.com/JanHerca/urantiapedia) 中创建一个包含每种语言脚注的 JSON 文件。
- 在 GitHub 上添加了《尤兰提亚之书》的英文版，并附有 JSON 格式的脚注。
