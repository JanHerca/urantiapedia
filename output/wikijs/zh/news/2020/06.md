## :sparkles: 新内容

- 网站尚未上线。内容部分，我们将继续直接上传至 [GitHub](https://github.com/JanHerca/urantiapedia)。
- 《尤兰提亚之书》的文件已生成 JSON 格式和 Wikitext 格式。有关这两种格式的信息，请参阅下文。
- 我们继续在《圣经》和《尤兰提亚之书》以及 [Paramony](https://urantia-book.org/urantiabook/paramony/) 之间添加交叉引用，这是由该书读者 Duane Faw 创建的现有列表。
- 目前所有内容仅提供西班牙语版本。

## :wrench: Urantiapedia 工具

- 考虑到_Urantiapedia Tools_最终将具有的复杂性，我们决定将代码从 C# 迁移到 Javascript，并使用 [Electron](https://www.electronjs.org/) 作为桌面执行环境。使用 Javascript 这种非常快速的编码语言，我们将能够简化所有开发流程，而且由于它是 Web 兼容的代码，将来可以在网页上重复使用。
- 添加了从《尤兰提亚之书》的 [LaTeX](https://en.wikipedia.org/wiki/LaTeX) 格式转换器到中间 [JSON](https://en.wikipedia.org/wiki/JSON) 格式，这将更方便地处理在其中添加的交叉引用。
- 添加了_The Urantia Book_从JSON格式到Wikitext格式的转换器。这种格式是 Wikipedia 软件 [Mediawiki](https://www.mediawiki.org/wiki/MediaWiki) 使用的格式，我们也将在 Urantiapedia 上使用。该转换器还在该过程中添加了 Paramony 脚注。
