## :sparkles: 新内容

- :blue_book: **《尤兰提亚之书》**：一些细微的印刷错误已得到纠正，并已开始着手将文献作者添加到索引中。未来几个月，我们计划在Urantia百科上发布的《尤兰提亚之书》副本尽可能与印刷版保持一致，使用相同的索引、封面，并在所有包含文献的页面上清晰地显示所有版权声明。
- :page_with_curl: **文章**：我们正在修复《The Fellowship Herald》英文版文章中的错误。我们还会继续将本杂志中的文章翻译成西班牙语。这项工作仍在进行中，未来将扩展到更多英文杂志和内容。

## :computer: GitHub 上的项目

目前在 GitHub 上，如[帮助](/zh/help)中所述，我们有两个项目：
- [urantiapedia](https://github.com/JanHerca/urantiapedia)：包含主项目，其中有一个名为 *Urantiapedia Tools* 的应用程序（“app”文件夹），还有一些用于存储信息的输入文件（“input”文件夹），以及存储所有网站内容的主输出文件夹（“output”文件夹）。
- [urantiapedia-backup](https://github.com/JanHerca/urantiapedia-backup)：包含同步到网站的先前输出文件夹 (`ouput`) 的副本。

到目前为止，*Urantiapedia Tools* 应用程序代码位于主项目中。上个月，我们开始将其迁移到位于 [urantiapedia-tools](https://github.com/JanHerca/urantiapedia-tools) 的独立项目。此举旨在赋予该应用程序独特的身份，使其能够快速发展，并通过将其迁移到更现代化的框架（例如 [Vue](https://vuejs.org/) 和 [Quasar](https://quasar.dev/)）来实现现代化。我们希望该应用程序在未来几个月内能够获得更多功能。我们想在与云数据库连接、在线翻译服务和人工智能管理相关的主题上探索许多新想法。有了更先进的应用程序，我们将能够更快、更高质量地创建 Urantiapedia 内容。目前，我们正在迈出迁移现有功能的第一步。:ok_hand: :+1:
